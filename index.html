University Work

Below are examples of some of the projects I have created throughout my time at University. Hover your mouse over the images to reveal more information. Some of projects also have more documentation and videos if you select the image title and follow the link.

Showreel

A horror themed showreel I created in my first year of university. I developed this using After Effects as a showcase of my skills.

Face Your Fears

This is my Final Year Project where I created a virtual reality simulation in which a person can become immersed in the subjective experiences of claustrophobia.

Dissertation

My dissertation on The Creative Potential of Virtual Reality in the Treatment of Phobias. This study examines the creative process of using virtual reality as a form of phobia treatment.

Lava Lamp

An interactive lava lamp coded with Maya Python

Fireflies

Fireflies is a visually dynamic installation that focuses on immersive interaction. The audience can interact with the characteristics of the projected fireflies using a projected Kinect system.

The Can

A realistic short animation of a falling can, created using Maya.

Explosion

A VFX experiment using Maya particle systems.

Pawpoint

A web and mobile application that links to a light-up dog tracking collar. Using this website the user can track their dogs location, set up a dog profile and change the light colours on the collar.

Lighthouse

This is a visually interactive application created using three.js. When the mouse is clicked, it cycles through the time of day, changing the appearance of the environment.
